# Deployment Checklist

Use this checklist to ensure your website is ready for deployment.

## Pre-Deployment Checklist

- [ ] All code changes are saved
- [ ] Website works correctly in local development (`npm run dev`)
- [ ] Build completes successfully (`npm run build`)
- [ ] No console errors in browser
- [ ] All images and assets load correctly
- [ ] Navigation links work properly
- [ ] Mobile responsive design works
- [ ] Contact forms and links work

## GitHub Setup Checklist

- [ ] GitHub account created/accessed (username: `sumanstiss`)
- [ ] GitHub Desktop installed
- [ ] New repository created on GitHub
- [ ] Repository is set to **Public** (required for free GitHub Pages)
- [ ] Local project connected to GitHub repository
- [ ] All files committed and pushed to GitHub

## GitHub Pages Configuration Checklist

- [ ] GitHub Pages enabled in repository Settings
- [ ] Source set to **GitHub Actions**
- [ ] GitHub Actions workflow file exists (`.github/workflows/deploy.yml`)
- [ ] First deployment triggered (push to main branch)
- [ ] Deployment completed successfully (check Actions tab)

## Post-Deployment Checklist

- [ ] Website URL accessible: `https://sumanstiss.github.io/[repository-name]/`
- [ ] All pages load correctly
- [ ] Images and assets load
- [ ] Navigation works
- [ ] Links work (internal and external)
- [ ] Mobile view works
- [ ] Contact information is correct
- [ ] Google Maps embed works
- [ ] Social media links work

## Testing Checklist

Test these on the live website:

- [ ] Home page loads
- [ ] Navigation menu works
- [ ] All pages accessible (Who We Are, What We Do, etc.)
- [ ] Projects page displays correctly
- [ ] Contact/Location page shows map
- [ ] Email links work (`mailto:` links)
- [ ] LinkedIn link works
- [ ] Mobile menu works
- [ ] Footer links work
- [ ] No 404 errors
- [ ] No console errors

## Common Issues to Check

- [ ] Base path is correct (automatically handled by GitHub Actions)
- [ ] All assets (images, fonts) load correctly
- [ ] Routes work (React Router)
- [ ] No broken links
- [ ] CSS styles applied correctly
- [ ] JavaScript animations work
